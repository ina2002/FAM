FAM 输入—自动输出网页原型

先录入专家、指标、方案，再自动生成输入表。填完专家权重、指标排序、方案排序和可选比值信息后，点一次“自动求解”，页面会直接输出方案排序、层级、指标权重和局部权重。

当前版本已按完整排序-分层模型接入 x_{kr}、ã_r、y_r、φ、λ_r，并增加了 η、min_separations 和 layer_guard 来减少模型无意义地优先判同层。
